# `.github` — Fiducial Mesh org defaults

This repository holds **organization-wide defaults** for [github.com/fiducial-mesh](https://github.com/fiducial-mesh). GitHub applies these to every repo in the org that does not define its own.

| Path | What it is |
|------|------------|
| `profile/README.md` | The org **profile page** (shown at the top of the org's GitHub landing page) |
| `.github/PULL_REQUEST_TEMPLATE.md` | Default pull-request template |
| `.github/ISSUE_TEMPLATE/` | Default issue forms (spec gap, bug) + chooser config |
| `CONTRIBUTING.md` | How to contribute — the spec-adherence + review discipline |
| `SECURITY.md` | Security policy + private vulnerability reporting |
| `CODE_OF_CONDUCT.md` | Contributor Covenant |

A repo that needs different behavior overrides any of these by providing its own copy.
